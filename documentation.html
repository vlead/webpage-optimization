1 Introduction

The main goal of this project is to test the various runtime metrics collections used for continuous system performance, quality and availability purposes and to further improve the availabilty of web pages. This is a performance comparison of many web application frameworks executing fundamental tasks such as client requets, server responses and rendering of web pages.
